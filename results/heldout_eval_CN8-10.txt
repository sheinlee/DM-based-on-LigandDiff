Held-out generative evaluation of the single-ligand DM (DM-based-on-LigandDiff) models.

Protocol: 40 held-out Ln seeds, CN 8-10 filter, save-all generation (generate_saveall, n_samples=8),
xtb GFN-FF relaxation, then per-ligand Validity / Uniqueness / Novelty + complex-level Yield +
whole-complex topology Recall. SAME held-out reference (Ln_data_new/val), SAME novelty reference
(ref_Ln: 1478 SMILES / 5624 WL graphs), SAME metric code as the coordination-site-conditioned Multi
model -> every row is directly comparable.

Two metric algorithms are reported:
  OUR strict metric : RDKit rdDetermineBonds (charge 0,+-1,+-2) + SanitizeMol per ligand, RELAXED geom.
  LITERATURE metric : OpenBabel build_mol + SanitizeMol (validity), GetMolFrags==1 (connectivity),
                      canonical-SMILES uniqueness, train-membership novelty; RAW geom; the same
                      molSimplify whole-complex ligand split as the multi-LigandDiff lit-metric.
                      (This reproduces how LigandDiff scores its own numbers.)

Models:
  ep276    = the connectivity-fine-tuned checkpoint (dconn3_from274_conn_epoch=276), trained on the
             larger CN 4-12 Ln_data_new (14,478); this IS the checkpoint used across the README.
  transfer = a staged TM-pretrain -> Ln-finetune run (ep134).

READING
- The connectivity fine-tune matters: ep276 (Validity 62.9%, Yield 10.0%) clearly beats its ep274
  base (59.8% / 4.9%) -- yield roughly doubled, reaching ~= the coordination-site-conditioned Multi
  (9.9%). Bigger data + connectivity fine-tuning closes the YIELD gap.
- Multi still leads overall: Validity 68.0% and Novelty 95.6% (our metric) remain ahead -- telling
  the model WHERE the coordination sites are helps beyond data / capacity / transfer.
- Staged TM->Ln transfer (ep134) underperforms (Validity 54.0%, lowest fidelity 0.473); undertrained.
- Under the literature algorithm the models reach novelty ~96% (in the literature band); the real gap
  is connectivity (~75% vs 96%), driven by the harder f-block domain and the fragile connectivity
  metric for coordination ligands.
- All models: 0% exact topology recall (consistent with the whole de novo coordination-complex field).
